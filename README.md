# R package: RUVSeq

This is the developer version of the Bioconductor package [RUVSeq](http://www.bioconductor.org/packages/devel/bioc/html/RUVSeq.html). Install in R as:

```r
source("http://bioconductor.org/biocLite.R")
biocLite("RUVSeq")
```
